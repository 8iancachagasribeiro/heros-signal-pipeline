mcPHASES IS NOT REDISTRIBUTED HERE.

It is credentialed-access PhysioNet data (DOI 10.13026/zx6a-2c81) governed by a data
use agreement that does not permit redistribution.

To reproduce the empirical analyses (manuscript section 7):
  1. Obtain PhysioNet credentialed access and accept the DUA.
  2. Download mcPHASES v1.0.0 and unzip it.
  3. Run:  python mcphases_analyses.py --data-dir /path/to/mcphases-...-1.0.0

The SIMULATION analyses (sections 3-6, 8) require NO data and run standalone.
